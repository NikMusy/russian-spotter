# Русский споттер

Споттер и инженер на радио для **Le Mans Ultimate** и **F1 25**, на русском.
Живой голос, а не синтез: фразы наговариваются в микрофон, программа их
склеивает и подаёт как командное радио — с полосой 300–3200 Гц, шипением
эфира и щелчком тангенты.

```
«сегодня едем, Ле-Ман, гонка»
«слева» · «с двух сторон» · «чисто»
«до машины впереди два и четыре секунды»
«полная жёлтая по трассе, сбрось скорость»
«гиперкар сзади, пропусти»
```

Две программы, обе с окном:

- **`recorder.exe`** — наговорить банк фраз
- **`spotter.exe`** — сам споттер, слушает игру и говорит

## Быстрый старт

1. `pip install -r requirements.txt`
2. `python -m spotter` — окно споттера
3. Выбери сим и голос, жми **СТАРТ**, заходи в игру

Готовые exe собираются одной командой — см. «Разработка» ниже.

## Голос: свой или чужой

В `voicepacks/` лежат готовые банки. Выбери голос в окне — и споттер
заговорит сразу, ничего не записывая.

Дальше можно наговаривать свои фразы через `recorder.exe`: **своя запись
всегда в приоритете**, а то, что не записал, берётся из пака. Так банк
собирается по кусочку, и споттер работает с первого дня.

- `sounds/` — твои записи (в репозиторий не входят)
- `voicepacks/nikolay/` — готовый банк, 301 фраза

Хочешь поделиться своим — положи папку в `voicepacks/` и присылай PR.

## 1. Записать фразы

Запусти `recorder.exe`.

Он показывает фразу — жмёшь **ПРОБЕЛ**, говоришь, жмёшь **ПРОБЕЛ** ещё раз.
Файл сохраняется сам, с правильным именем. Тишину по краям он обрезает,
громкость выравнивает — подгадывать момент не надо, жми пробел, вдохни,
говори.

- **ПРОБЕЛ** — начать/остановить запись
- **P** — прослушать, **←** / **→** — назад/вперёд
- сверху выбирается сим, микрофон, и нужен ли мат
- галка «только необходимые» оставит минимум (170 файлов вместо 233)

Записывать можно частями. Споттер работает с тем, что уже есть, а
незаписанное молча пропускает. Рекордер сам открывается на первой
незаписанной фразе, так что можно наговорить полчаса и вернуться завтра.

**Начни с группы СПОТТЕР** — 11 фраз, которые реально спасают в гонке.

## 2. Включить телеметрию в игре

F1 25 → Настройки → Телеметрия:

| Параметр | Значение |
|---|---|
| UDP Telemetry | ON |
| UDP Broadcast Mode | OFF |
| UDP IP Address | 127.0.0.1 |
| UDP Port | 20777 |
| UDP Send Rate | 60 Hz |
| **UDP Format** | **F1 25** |

UDP Format обязан быть **F1 25**. Если стоит «F1 25: 2026 Season Pack» —
формат пакетов другой, и споттер их проигнорирует.

Если игра на другом ПК или консоли — впиши там IP компьютера со споттером.

## 3. Запустить

`spotter.exe` → **СТАРТ**. Порядок запуска с игрой любой.

Кружок рядом с кнопкой: серый — стоит, жёлтый — слушает, зелёный — есть
телеметрия. В окне «Радио» видно всё, что споттер сказал.

Настройки живут в `config.json` рядом с exe и сохраняются сами.

## Симы

| Сим | Файлов записать | Телеметрия |
|---|---|---|
| **F1 25 / F1 24** | 265 | **работает** (UDP) |
| **Le Mans Ultimate** | 287 | **работает** (разделяемая память) |
| Assetto Corsa Competizione | 273 | нет адаптера |
| Automobilista 2 / Project CARS 2 | 281 | нет адаптера |
| iRacing | 289 | нет адаптера |

Все фразы написаны от лица споттера, который говорит тебе («твои шины
умерли»), а не от пилота — записывай их как обращение к гонщику.

Трассы для LMU — весь календарь WEC и ELMS: Ле-Ман, Себринг, Фудзи,
Портимао, Поль Рикар, Муджелло, Арагон, Спа, Монца, Имола, Бахрейн,
Интерлагос, Остин, Катар, Сильверстоун, Барселона. При заходе в сессию
споттер объявляет: «сегодня едем, Ле-Ман, гонка».

Фразы в банке общие, плюс свои для каждого сима: эндуранс — трафик
классов, прототип сзади, смена пилота, фары, полная жёлтая; iRacing —
инциденты; F1 — DRS. Выбор сима в рекордере показывает только то, что
этому симу нужно.

### Le Mans Ultimate

**Ничего настраивать не надо** — ни плагинов, ни портов. LMU публикует
данные сама, споттер их читает.

Выбери в списке «Le Mans Ultimate», жми СТАРТ, заходи в сессию. Порядок
запуска любой: споттер ждёт игру и подхватывает её сам.

Проверить связь: **`lmu_check.exe`** — запусти, когда сидишь в сессии. Он
покажет трассу, погоду, список машин и скажет, всё ли читается верно.

В LMU споттер говорит «полная жёлтая по трассе», а не «машина
безопасности» — там это Full Course Yellow.

Читается через официальный интерфейс Studio 397: заголовки лежат прямо в
игре, в `Support/SharedMemoryInterface/`. Сторонний плагин
(rF2SharedMemoryMapPlugin) не нужен.

## Что говорит

- **Споттер**: слева / справа / с двух сторон / чисто. Прерывает всё
  остальное.
- **Флаги**: жёлтый, синий, красный, зелёный.
- **Сейфти-кар**: полный и виртуальный, рестарт.
- **Погода**: дождь начался/кончился, трасса сохнет, температура трассы, и
  прогноз — «дождь через 10 минут, вероятность 70 процентов».
- **Шины**: грей, перегрев, износ, прокол.
- **Топливо**: экономь / хватает / в обрез.
- **Повреждения**: крыло, днище.
- **Пит**: включи лимитер (спасает от штрафа), бокс.
- **Гонка**: позиция, отрыв, зона DRS, последний круг, личный рекорд.
- **Штрафы**: границы трассы, пять секунд.

Числа склеиваются из кусков: «до машины впереди» + «два» + «и» + «четыре» +
«секунды».

## Эффект рации

Голос обрабатывается под настоящее командное радио: полоса 300–3200 Гц
(ни низа, ни воздуха), поджатая динамика с лёгкой перегрузкой, шипение
эфира, щелчок тангенты в начале и затухающий хвост в конце.

Записывать надо **чисто и близко к микрофону** — эффект накладывается
поверх при воспроизведении, сами файлы остаются нетронутыми. В рекордере
есть кнопка **«Как в рации (R)»** — послушать, как фраза прозвучит в игре.

Галка «Рация» и ползунок «Шипение» — в окне споттера. Тонкая настройка в
`config.json`:

```json
"radio": {
  "enabled": true,
  "low": 300,        // нижний срез, Гц
  "high": 3200,      // верхний срез, Гц
  "drive": 2.2,      // насыщение: больше - грязнее
  "noise": 0.055,    // уровень шипения
  "tail": 0.14,      // хвост шума после фразы, сек
  "lead": 0.05,      // шум перед фразой
  "click": 0.35,     // щелчок тангенты, 0 - выключить
  "crackle": 0.15    // вероятность треска в фразе
}
```

Хочешь совсем грязный эфир — подними `drive` до 4 и `noise` до 0.12.
Нужен чистый голос — `"enabled": false` или `spotter.exe --no-radio`.

## Мат

У 32 фраз есть дубль с матом (`*__hard.wav`). Если он записан и галка
«Мат» стоит — играет он. Снял галку — вернутся обычные, файлы останутся на
месте.

## Если не работает

- **Кружок жёлтый, «жду телеметрию»** — игра не шлёт пакеты. Проверь UDP
  Telemetry: ON, порт и IP.
- **Зелёный, но молчит** — фразы не записаны. Смотри счётчик «записано N
  из 233» и строки `[нет файла] X.wav` в логе.
- **Игра шлёт, а кружок жёлтый** — скорее всего UDP Format стоит «2026
  Season Pack» вместо «F1 25».
- **«Не стартует: ... address already in use»** — порт занял SimHub или
  F1Laps. Включи в игре UDP Broadcast Mode, тогда получат обе программы.

```
python tools\test_proximity.py     тесты геометрии споттера

```

Проверить споттера без игр: запусти `spotter.exe`, выбери сим, СТАРТ, потом
`simulate.py` (для F1) или `fake_lmu.py` (для LMU) — они разыграют гонку:
соперники слева и справа, флаги, дождь, прокол, пит-лейн.

Сборка exe:

```
python -m PyInstaller --onefile --name spotter --windowed --icon tools\icon.ico --collect-submodules spotter --paths . --distpath . --workpath build --specpath build -y tools\spotter_entry.py
python -m PyInstaller --onefile --name recorder --windowed --icon tools\icon.ico --collect-all sounddevice --collect-submodules spotter --paths . --distpath . --workpath build --specpath build -y tools\recorder_entry.py
```

`--icon` требует абсолютный путь, иначе PyInstaller ищет иконку
относительно `--specpath`.

### Про заголовки Studio 397

Структуры LMU написаны по официальному SDK, который лежит внутри самой
игры: `Le Mans Ultimate/Support/SharedMemoryInterface/`. Сами заголовки в
репозиторий **не входят** — они проприетарные, и их лицензия запрещает
распространение. `spotter/sims/lmu_structs.py` — это описание формата на
Python, написанное по ним.

Главная тонкость: `#pragma pack(push, 4)` действует только внутри
`InternalsPlugin.hpp` и закрывается там же. Структуры `SharedMemory*`
объявлены уже после него, то есть с обычным выравниванием. Если поставить
`pack = 4` всем подряд, массив машин съезжает на 4 байта: имена пилотов
читаются пустыми, координаты превращаются в 1e152.

### Как устроено

- `spotter/udp/packets.py` — разбор бинарных пакетов. Размеры структур
  сверены со спецификацией «Data Output from F1 25»; пакет неожиданного
  размера отбрасывается, а не парсится в мусор.
- `spotter/state.py` — состояние гонки, собранное из пакетов.
- `spotter/rules/` — правила. Читают состояние, зовут `say(...)`.
- `spotter/rules/proximity.py` — геометрия споттера: позиции соперников
  переводятся в систему координат игрока через векторы направления из
  Motion-пакета.
- `spotter/audio/player.py` — очередь с приоритетами.
- `spotter/audio/radio.py` — эффект рации. FIR-фильтр на numpy со свёрткой
  через FFT: ядро длиной 1301 отсчёт нужно, чтобы честно срезать низ на
  300 Гц, а прямая свёртка таким ядром тормозила бы первый вызов споттера.
- `spotter/audio/phrases.py` — банк фраз. Единственный источник правды:
  id фразы == имя wav-файла.
- `spotter/gui.py`, `spotter/recorder_gui.py` — окна на tkinter.

Плеер спрашивает у pygame фактический формат микшера (`get_init()`) и
подстраивает буфер: моно запросто открывается как стерео, и тогда
`make_sound` требует двумерный массив.

#### Про LMU

- `spotter/sims/lmu_structs.py` — структуры разделяемой памяти, один в один
  с `Support/SharedMemoryInterface/` из самой игры. В заголовках стоит
  `#pragma pack(push, 4)`, поэтому у всех структур `_pack_ = 4`: с
  выравниванием по умолчанию поля разъезжаются и читается мусор.
- `spotter/sims/lmu.py` — чтение памяти и перевод в общий `GameState`,
  дальше работают те же правила, что и у F1.

Память открывается через `OpenFileMapping`, а **не** через
`mmap(-1, tagname=...)`: последний, не найдя объект, молча создаёт свой
пустой — и споттер считает, что подключился к игре, читая нули.

Оси у LMU левосторонние, а локальные у машины — `+x` налево, `+z` назад
(так в `InternalsPlugin.hpp`). При переводе в наши «вперёд» и «вправо»
знаки инвертируются, иначе лево и право меняются местами.

Читаем без блокировки: спинлок LMU держится на
`InterlockedCompareExchange`, которого из Python не вызвать. Порванный кадр
теоретически возможен, но данные обновляются 60 раз в секунду.

Чтобы добавить сим: написать адаптер, который наполняет `GameState`, —
правила и звук трогать не придётся.
