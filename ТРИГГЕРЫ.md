# Когда споттер что говорит

Полная карта триггеров. Все пороги правятся в исходниках — файл указан у
каждой группы.

## Приоритеты

Кто кого перебивает:

| Уровень | Что | Поведение |
|---|---|---|
| 0 | споттер (слева/справа), прокол | **Обрывает** любую фразу на полуслове |
| 1 | флаги, бокс, лимитер, дождь | Играет раньше обычных |
| 2 | позиция, отрывы, шины, классы | Обычная очередь |
| 3 | температура, топливо в норме | Фон, если нет ничего важнее |

Споттерские вызовы не копятся в очереди: если пришёл новый, старый
выбрасывается — важно то, что происходит **сейчас**, а не полсекунды назад.

## Споттер — машина рядом

`spotter/rules/proximity.py`

Считается каждый кадр (60 раз в секунду).

| Фраза | Когда |
|---|---|
| «слева» / «машина слева» | Соперник появился слева: перекрытие вдоль ±6.4 м, вбок 1.2–9 м |
| «справа» / «машина справа» | То же справа |
| «с двух сторон» | Есть и слева, и справа |
| «всё ещё рядом» | Держится рядом дольше 3.5 сек, повторяется каждые 3.5 сек |
| «чисто» / «свободен» | Отъехал и не возвращается 0.55 сек |

Молчит когда:
- скорость ниже **45 км/ч** (гараж, пит-лейн, старт);
- ты в пит-лейн;
- соперник в пит-лейн, в гараже или неактивен;
- «чисто» не скажет, если до этого не сказал «слева»/«справа».

Гистерезис: пока кто-то рядом, границы шире (вдоль 8.7 м, вбок 11 м) — на
кочках и в поворотах соперник не «мигает».

## Трасса и сессия

`spotter/rules/track.py`

| Фраза | Когда |
|---|---|
| «сегодня едем» + трасса + тип сессии | Один раз при заходе в сессию |
| «квалификация, один круг — выложись» | Сессия — квалификация |
| «практика, никуда не спеши» | Сессия — практика |
| «поехали, удачи» | Сессия — гонка |

Трасса определяется по id (F1) или по названию (LMU). Не узнал — скажет
просто «трасса». Повторно за сессию не объявляет; сменишь трассу или тип —
объявит заново.

## Погода

`spotter/rules/weather.py`

| Фраза | Когда |
|---|---|
| «трасса сухая» / «по траектории мокро» | Один раз при заходе |
| «температура трассы N градусов» | Один раз при заходе |
| «трасса холодная» / «горячая» | Ниже 22 °C / выше 45 °C |
| «начинается дождь» + «подумай про промежуточные» | Погода стала мокрой |
| «сильный дождь» + «пора на дождевые» | Дождь усилился |
| «дождь заканчивается» + «трасса сохнет» + «можно на слики» | Высохло |
| «дождь ожидается через N минут» | Прогноз на ближайшие 20 минут |
| «вероятность дождя N процентов» | Следом за прогнозом, если известна |

Прогноз — **только F1**: LMU наружу его не отдаёт. Не чаще раза в 90 сек.

## Шины и тормоза

`spotter/rules/car.py`

| Фраза | Когда | Сколько раз |
|---|---|---|
| **«прокол»** | Повреждение шины ≥ 80% | сразу |
| «грей шины» | Ниже 70 °C и быстрее 60 км/ч | **не больше 2 раз за стинт**, не чаще 50 сек |
| «шины в рабочей температуре» | 85–105 °C | один раз, и про холод больше не заикается |
| «шины перегреваются» | Выше 115 °C | не больше 3 раз за стинт, не чаще 45 сек |
| «шины на исходе» | Износ 70% | один раз на комплект |
| «шины убиты, надо в бокс» | Износ 85% | один раз на комплект |

Счётчики обнуляются на выезде из пита — новый стинт, резина другая.

Лимит на «грей шины» стоит не просто так: если трасса холодная, резина
может вообще не догреться до нормы, и без него споттер долбил бы одно и то
же весь заезд.

## Топливо

| Фраза | Когда |
|---|---|
| «экономь топливо» | Не хватает 0.15–0.6 круга |
| «топлива в обрез» | Не хватает больше 0.6 круга |
| «топлива хватает, можешь атаковать» | Запас больше круга |

Считается от **остатка гонки**, а не от бака. Каждое — один раз за гонку,
раньше 2-го круга молчит.

## Повреждения

| Фраза | Когда |
|---|---|
| «переднее крыло повреждено» | Повреждение ≥ 20% |
| «переднее крыло сильно повреждено, в бокс» | ≥ 60% |
| «заднее крыло повреждено» | ≥ 30% |
| «днище повреждено» | ≥ 40% |

Каждое — один раз, пока не починишь.

## Гонка

`spotter/rules/race.py`

| Фраза | Когда | Не чаще |
|---|---|---|
| «отличный обгон» / «тебя обошли» + «ты N-й» | Позиция изменилась | 4 сек |
| «до машины впереди N и N секунды» | Отрыв меньше 25 сек | 45 сек |
| «ты в зоне DRS» (F1, AMS2) | Отрыв меньше 1 сек | 20 сек |
| «ты в слипстриме» (LMU и эндуранс) | Отрыв меньше 0.7 сек | 15 сек |
| «плотно в мешке, готовь обгон» | Отрыв меньше 0.4 сек | 15 сек |
| «последний круг» | Остался круг | — |
| «осталось кругов N» | Осталось 2–5 | каждый круг |
| «половина дистанции» | Середина (от 10 кругов) | — |
| «личный рекорд» | Круг быстрее прошлого лучшего | — |

На первом круге позиции скачут — правило молчит. В пит-лейн тоже.

Числа склеиваются: «до машины впереди» + «два» + «и» + «четыре» +
«секунды».

## Штрафы

`spotter/rules/events.py`

| Фраза | Когда | Не чаще |
|---|---|---|
| «следи за границами трассы» | Счётчик предупреждений вырос | — |
| «штраф, проезд по пит-лейн» | Выписан drive-through | 5 сек |
| «штраф, стоп энд гоу» | Выписан stop&go | 5 сек |
| «пять секунд штрафа» | Штраф временем | 5 сек |
| «на тебе висит штраф» | Счётчик штрафов вырос | 6 сек |
| «отбывай штраф в этом круге» | Штраф не отбыт | 45 сек, пока висит |
| «штраф отбыт, поехали» | Неотбытых не осталось | — |

Тип штрафа F1 присылает прямо в событии, поэтому там слышно точно:
стоп-энд-гоу это или проезд. **LMU отдаёт только счётчик** — отличить
нельзя, поэтому звучит общее «на тебе висит штраф».

## Классы машин (эндуранс)

`spotter/rules/classes.py` — работает там, где сим отдаёт классы (**LMU**).

| Фраза | Когда | Не чаще |
|---|---|---|
| «гиперкар сзади, пропусти» | Гиперкар ближе 45 м сзади, ты медленнее | 12 сек |
| «эл эм пэ два сзади» | То же для LMP2 | 12 сек |
| «сзади быстрый класс, пропусти» | Любой класс быстрее сзади | 12 сек |
| «джи ти три впереди, обгоняй» | GT/GTE ближе 60 м впереди, ты быстрее | 12 сек |
| «трафик впереди» | Любой класс медленнее впереди | 12 сек |
| «ты лидер класса» | Никого из твоего класса впереди | 30 сек |

Кто быстрее: гиперкар → LMP2 → LMP3 → GT3/GTE. Свой класс не считается —
это уже борьба, а не трафик. Смотрит только по своему ряду (в пределах
12 м вбок), скорость выше 40 км/ч.

## Мемы

`spotter/audio/phrases.py`, карта `MEME_FOR`.

Мем **не отдельный триггер**: он изредка заменяет обычную фразу. Шанс —
25% (`meme_chance` в `config.json`), галка «Мемы» в окне.

Все мемы — от лица споттера, который говорит **тебе** (не слова пилота).
У каждого есть матерный дубль: 17 мемов = 34 файла.

| Мем | Матерный дубль | Заменяет |
|---|---|---|
| «тутутуту, Макс Ферстаппен» | «тутутуту, блядь, Макс Ферстаппен ебаный» | отличный обгон, красиво |
| «ты просто Шумахер» | «ебать ты Шумахер ебаный» | быстрейший круг, личный рекорд |
| «понял, не мешаю, ты знаешь что делаешь» | «да похуй, делай что хочешь» | «всё ещё рядом», «соберись» |
| «твои шины умерли, Льюис» | «шины твои сдохли нахуй, Льюис» | шины на исходе / убиты |
| «мотор звучит как джи пэ два» | «мотор у тебя как у ебаной джи пэ два» | серьёзные повреждения |
| «давай, залетай туда» | «давай, блядь, залетай нахуй» | обгон, «жми» |
| «куда ты лезешь, олень» | «куда ты лезешь, долбоёб» | тебя обошли, контакт |
| «ты плывёшь, как на лодке» | «ты плывёшь нахуй, как на лодке» | мокрая траектория, ливень |
| «чемпион мира» | «чемпион мира, блядь, ебать ты дал» | победа, быстрейший круг |
| «ты не Райкконен, успокойся» | «ты не Райкконен нахуй, успокойся» | круг не засчитан |
| «ну ты и валенок» | «ну ты и валенок ебаный» | круг не засчитан, границы |
| «красиво, плавный оператор» | «красиво, блядь, плавный оператор» | хороший круг, личный рекорд |
| «педаль в пол и не думай» | «педаль в пол нахуй, не думай» | «жми, сейчас твой шанс» |
| «изи катка» | «изи катка, блядь» | обгон, красиво |
| «это дно, ты пробил дно» | «это пиздец, ты пробил дно» | машина в хлам, топливо в обрез |
| «улыбайся, ты в телевизоре» | «улыбайся, блядь, ты в телевизоре» | сессия завершена |
| «бокс бокс бокс, я сказал» | «бокс бокс бокс, блядь, я сказал» | бокс в этом круге |

Мем не записан — играет обычная фраза. Ничего не потеряется.

Мат и мемы совмещаются: сначала выбирается мем, потом — есть ли у него
матерный дубль. Галка «Мат» снята — играет цензурный вариант мема.
